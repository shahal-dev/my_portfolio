# 多页面作品集 - 一个使用 Next.js 14 构建的现代化、响应式作品集网站，包含多个页面，包括博客系统、项目展示和关于页面。

#### 预览

 - [在线演示](https://themewagon.github.io/multiplepage-portfolio/)

#### 下载
 - [从 ThemeWagon 下载](https://themewagon.com/themes/multiplepage-portfolio/)

## 开始使用

1. 克隆仓库
```
git clone https://github.com/themewagon/multiplepage-portfolio.git
```
2. 安装依赖
```
npm i
```
3. 运行开发服务器：

```bash
npm run dev
# 或
yarn dev
# 或
pnpm dev
# 或
bun dev
```
